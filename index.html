    Installing the plugin

    1. Include jQuery, mosaicPhoto script and styles:

      <script src="//ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script src="js/jquery.mosaicPhoto.js"></script>
<link rel="stylesheet" href="css/mosaicPhoto.css">

    2. Create the object with arrays of images:

      var images = {
	'cat1' : [
		["/img/cat1/1.jpg", "/img/cat1/big/1.jpg"],
		 "/img/cat1/2.jpg",
		 "/img/cat1/3.jpg",
		["/img/cat1/4.jpg", "/img/cat1/big/4.jpg"],
		 "/img/cat1/5.jpg",
		 "/img/cat1/6.jpg",
		 "/img/cat1/7.jpg",
		["/img/cat1/8.jpg", "/img/cat1/big/8.jpg"],
		 "/img/cat1/9.jpg",
		 "/img/cat1/10.jpg"
	],
	'cat2' : [
		["/img/cat2/1.jpg", "http://site.com/news"],
		["/img/cat2/2.jpg", "http://site.com/about"],
		["/img/cat2/3.jpg", "http://site.com/contacts"],
		["/img/cat2/4.jpg", "http://site.com/feedback"],
		["/img/cat2/5.jpg", "http://site.com/catalog"]
	]
	};

      You can have several categories of images and each image can have a link to the full image or to some other URL. You need at least one category for script to work.

    3. Create the container element

      <div id="previewContainer" class="mosaicContainer"></div>

    4. Finally launch the mosaicPhoto script

      jQuery(document).ready(function($){
  $("#previewContainer").mosaicPhoto({
      images: images
  });
});

    Default settings

    You can pass object of settings when initializing or reinitializing mosaicPhoto.

    Here are the defaults:

    mosaicPhoto.defaults = {
  rows: 3, /* number of rows with images */
  cols: 8, /* number of columns with images*/
  ratio: 0.8, /* amount of filled spots. 0 - none, 1 - all spots are filled with images, but no more than of maxImages */
  maxImages: 24, /* maximum this number of images will be shown. */
  dynamic: true, /* if set to true, columns and rows are set dynamically depending on container element. overrides 'rows' and 'cols' settings */

  activeClass: 'mosaicItem', /* CSS class for mosaic elements*/
  fadeTime: 750, /* fading time, when refreshing container */
  auto: false, /* enable auto refreshing */
  timer: 3000, /* time for autorefreshing in milliseconds */

  preload: true, /* preloading images is enabled */

  images: {}, /* object with arrays of images. must contain at least 1 category. */

  listeners: '', /* string representing listener objects container. i.e. "#menu a".
    'listenersEvents' are delegated on those. Tags must contain 'rel' attribute with a value of correspoding images category.
    (<div id="menu"> <a href="#" rel="cat1">category 1</a> <a href="#" rel="cat2">category 2</a></div>) */
  listenersEvents: 'click', // event for listeners

  showAllCallback: function () {
  }, // executed after mosaic has been filled with images
  listenerCallback: function () {
  } // executed after listener has finished its event
};

    All default settings are stored in mosaicPhoto.defaults object. You can change the default for all instances of mosaicPhoto by editing this object

    mosaicPhoto.defaults.preload = false; // disable preloading of images for all instances of mosaicPhoto

    Methods

    After initializing mosaicPhoto on an element you can call these methods:

    getOptions([string optionName])

    If optionName is set, returns it's value, else returns object with all options.

    $("#previewContainer").mosaicPhoto().getOptions('maxImages');  // 24

    setOptions(object options)

    Accepts object with new options. You need to use refresh() to see the effects of setOptions

    $("#previewContainer").mosaicPhoto().setOptions({ ratio: .3, auto: true, timer: 5000 }).refresh();
    // fills up to 30% of area with images and auto refreshes them every 5 seconds

    Trying to reinitialize already initialized objects also calls setOptions:

    $("#previewContainer").mosaicPhoto({ ratio: .3, auto: true, timer: 5000 }).refresh();
    // does the same as code above

    selectCategory(string categoryName)

    Images from categoryName will be shown on next refresh of mosaicPhoto. You need to use refresh() to see the effects of selectCategory

    $("#previewContainer").mosaicPhoto().selectCategory('cat2').refresh(); // shows images from cat2

    refresh()

    Refreshes mosaicPhoto container.

    $("#previewContainer").mosaicPhoto().refresh();

    destroy()

    Destroys mosaicPhoto instance and all data

    $("#previewContainer").mosaicPhoto().destroy();

    Static container

    var $cont2 =$("#previewStaticContainer");

    // initialize mosaicPhoto
    $cont2.mosaicPhoto({
      images: images2,
      dynamic: false,
      cols: 19,
      rows: 5,
      maxImages: 20,
      auto: true,
      ratio: .05, // only 5% of area are filled with images
      fadeTime: 200
    });



    //reinitilize with new fading time and new ratio
    $cont2.mosaicPhoto({
      ratio: .25, // 25% of area are filled with images now
      fadeTime: 450,
      timer: 1000
    }).refresh();

    // toggle animation with button element
    $('#toggleAnim').click(function(e){
      var curState = $cont2.mosaicPhoto().getOptions('auto'),
      newState = !curState;
      $(this).html(newState ? 'Stop animation' : 'Start animation');
      $cont2.mosaicPhoto().setOptions({'auto':newState});
      return e.preventDefault();
    });
